About The Project

How AI is transforming Airbnb pricing in India

What This Does

This project uses machine learning to predict optimal nightly prices for Airbnb listings across Indian cities. By analyzing property details, location, host information, and market trends, it provides data-driven pricing recommendations to help hosts maximize their earnings.

The Model
  • Algorithm: CatBoost Regressor
  • Features: 19 property, location & host attributes
  • Training Data: Thousands of Airbnb listings
  • Target: Log-transformed nightly price
  • Preprocessing: Ordinal encoding + StandardScaler

How It Works
01

Data Collection

Airbnb listing data is collected and preprocessed — missing values are imputed, categorical features are encoded, and numeric features are normalized.

02

Model Training

Multiple algorithms are evaluated (Linear Regression, Random Forest, Gradient Boosting, CatBoost). The best performing model is saved for predictions.

03

Web Application

Flask serves the model through a modern web interface with real-time exchange rates, interactive maps, and analytics dashboards.
